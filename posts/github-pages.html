Using GitHub Pages

Posted for Module 23 activity

Summary

This site is hosted using GitHub Pages. I created a repository with HTML files (including an index page), added a shared CSS file for styling, and then enabled Pages in the repository settings to publish the site.

What I did

  1. Created a repository and added index.html.
  2. Added additional pages (About, Projects, Posts).
  3. Added a shared stylesheet and basic accessibility features (skip link, focus styles).
  4. Enabled GitHub Pages from the main branch root folder.

The result is a static website that can be shared using a public URL, and updated by committing changes to the repository.
